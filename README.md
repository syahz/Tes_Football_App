
# Simple Football App

Menggunakan ExpressJS untuk Backend nya dan React untuk Frontend nya


## Installation

Install be AND fe with npm

```bash
  cd be
  npm install

  cd fe
  npm install
```

Copy db.sql to your PC with Laragon or Phpmyadmin
